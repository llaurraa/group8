# ⚡ Olympus Dash: 眾神之路

## 🎮 遊戲簡介
這是一場從「凡人」晉升為「天神」的熱血冒險！
在華麗的 3D 世界中奔跑，穿越傳說中的希臘五大界域，享受即時生成的史詩音樂。

---

## 💡 設計動機 (為什麼要做？)
*   **體驗成神之路**：讓玩家親自體驗希臘神話的壯闊，而不只是看故事。
*   **拒絕單調**：打破跑酷遊戲場景重複的缺點，透過不斷變化的環境與音樂，讓每一分鐘都充滿新鮮感。

## 🎨 設計概念 (核心體驗)
*   **視覺饗宴**：使用現代 3D 技術，呈現發光的雷霆、深海的幽光與地獄的烈火。
*   **聽覺魔法**：音樂不是預錄的，而是程式碼即時「演奏」的。隨著遊戲越緊張，鼓聲就越激烈！
*   **風險與獎勵**：越危險的動作分數越高！鼓勵玩家挑戰極限，而不只是保守求生。

---

## ✨ 遊戲特點 (哪裡好玩？)

### 1. 🌍 五大世界，無縫切換
不用讀取畫面！隨著你越跑越遠，世界會自動改變：
*   ☁️ **奧林帕斯** (神聖殿堂)
*   🔥 **冥府深淵** (岩漿地獄)
*   🌊 **亞特蘭提斯** (深海遺跡)
*   ⚔️ **特洛伊戰場** (漫天黃沙)
*   🌌 **混沌虛空** (最終試煉)

### 2. 🎥 動態鏡頭運鏡
*   **🟢 安全區**：鏡頭拉遠，視野開闊，讓你盡情享受撿金幣的快感。
*   **🔴 危險區**：鏡頭瞬間拉近壓低，氣氛變得壓抑緊張，考驗你的反應速度！

### 3. 🛍️ 3D 沉浸式商店
買角色不再是看照片！你可以 360 度欣賞 **斯巴達戰神** 或 **天界信使** 的帥氣造型與裝備細節。

### 4. 🧠 策略跑酷
*   **體力制**：二段跳需要體力，不能無腦亂跳。
*   **連擊系統**：貼著障礙物閃避或滑鏟破壞物件可以累積 Combo，分數直接翻倍！

---

## 🎮 如何操作？
*   **⬅️ ➡️**：左右移動
*   **⬆️ (W)**：跳躍 (空中再按一次可二段跳)
*   **⬇️ (S)**：滑行 (躲避高處障礙)
*   **空白鍵**：開啟無敵盾牌 (需先購買道具)
